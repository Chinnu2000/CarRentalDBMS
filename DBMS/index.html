About the Project

Topic: Online Car Rental System

Our project is an online website that manages a car renting business. The system provides the user to rent a car as per their specifications like brand, cost, capacity, etc. It provides all functions including listing car information, booking, and payment. The admin can add a car, manage booking car and rent and also view feedback and enquiry. Users can view information of available cars, booking cars, easily get the car on rent and also give feedback and can do enquiry.
